📚 Cochrane Reviews searches Europe PMC filtered to the Cochrane Database of Systematic Reviews journal. It is driven by your primary (MeSH) query and merged/deduplicated with other sources.
👓 Scopus needs an Elsevier API key, entered in Settings (⚙ top right). Get one free at dev.elsevier.com.

The key works on your institution's network. Off-campus, you also need an InstToken — request one from Elsevier API Support using your institutional email.

Note: Scopus returns up to 25 results per search regardless of the per-database limit, due to Elsevier API restrictions.
🧪 ClinicalTrials.gov searches the AACT (Aggregate Analysis of ClinicalTrials.gov) public API for registered and unpublished clinical trials. Results include ongoing, completed, and terminated trials that may not yet appear in PubMed. Useful for identifying grey literature and publication bias. No API key required.
ℹ️ Two-stream search. The primary bar takes your full PubMed MeSH/boolean strategy and drives the MEDLINE-indexed sources (PubMed, Europe PMC, Cochrane). The secondary bar is auto-derived as a plain-keyword version (MeSH tags stripped) for the natural-language engines that don't understand MeSH. Edit the secondary freely — it won't be overwritten unless you click Re-derive.

3
Automated Quality Checklists
— run automatically on any paper with full text available
Once a paper's full text is retrieved (via PubMed Central or a free PDF), the app will automatically run quality checklists against the text. Each checklist item is keyword-scanned and pre-filled as a starting point — you can review and adjust each item manually inside the paper's expanded view. Checklists are matched to study type by default but can be changed per paper. The appropriate risk of bias tool is automatically selected and assessed alongside the matched quality checklist (e.g. RoB2 for RCTs, ROBINS-II for observational studies, Newcastle-Ottawa for case-control studies).
